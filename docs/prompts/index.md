# Prompt Kits

**Audience:** Faculty and students

Use these as operational kits, not just reading pages.

## What each kit does
- [AI Use Policy](ai-use-policy.md): sets declaration and verification-trace requirements for AI-assisted work.
- [Good Questions](good-questions.md): generates stronger notebook prompts for class, office hours, and feedback.
- [Evidence Check](evidence-check.md): validates claims before they become recommendations.

## Quick use sequence (weekly)
1. Start with [Good Questions](good-questions.md) to frame the decision.
2. Apply [Evidence Check](evidence-check.md) before finalizing conclusions.
3. Enforce [AI Use Policy](ai-use-policy.md) declaration and verification trace.

## Where these connect
- Student artifact: [Warrant Template](../warrant-template.md)
- Scoring system: [Reasoning Rubric](../rubrics/reasoning-rubric.md)
