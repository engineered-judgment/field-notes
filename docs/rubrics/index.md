# Rubrics

**Audience:** Faculty (with student transparency)

Rubrics are the scoring backbone for Warrant entries.

## How to use this section
1. Start with [Reasoning Rubric (v2)](reasoning-rubric.md).
2. Select the scoring mode for your phase (week 1, week 2, or full).
3. Calibrate on 3 sample entries before first live grading.
4. Publish criteria and weights to students before submission week.

## Why this matters
A shared rubric lowers grading variance, clarifies expectations, and makes adoption portable across courses.
